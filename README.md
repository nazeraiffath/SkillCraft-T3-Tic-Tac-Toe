# Tic Tac Toe Game 🎮
## SkillCraft Technology Internship - Task 3

Neon Tic Tac Toe is an interactive and visually attractive web-based game developed as part of the SkillCraft Technology Internship Task 3.

This project is built using **HTML, CSS, and JavaScript**. The game allows the user to play against the system where the player controls **X** and the system plays as **O**. The application includes an attractive neon interface, dynamic gameplay, score tracking, winner detection, and draw handling.

## Features ✨

- Interactive 3×3 Tic Tac Toe board
- Player vs System gameplay
- Automatic system moves
- Real-time turn updates
- Winner and draw detection
- Score tracking system
- Restart game functionality
- Neon-style attractive UI
- Responsive design


## Technologies Used 💻

- HTML5
- CSS3
- JavaScript

2. Open the `index.html` file in your browser

3. Start playing the game
